---- HOW TO USE WEB CALCULATOR ---

Use it just like how you would any in-person calculator.

The web calculator made for this project is a very basic calculator that does
basic arithmetic. 

See below for operators included.

Operators included in the Web Calculator:

1. (+) Addition
2. (-) Subtraction
3. (*) Multiplication
4. (/) Division

Other functionalities:

- (AC) To delete all inputted numbers
- (DEL) Serves as a backspace to delete last inputted number
